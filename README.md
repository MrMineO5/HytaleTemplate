# Hytale Plugin Template

> [!NOTE]
> The `java` branch contains the java template, switch to branch [`master`](https://github.com/MrMineO5/HytaleTemplate/tree/master) if you prefer to use Kotlin

A template for creating Hytale plugins, uses the [Hytale Gradle Plugin](https://github.com/MrMineO5/HytaleGradlePlugin) to pull in the Hytale dependency and provide tasks.

## Quickstart
1. Download the template
2. Open the project in your favorite IDE (I recommend IntelliJ IDEA)
3. Run the `runServer` gradle task
4. Join the local server
